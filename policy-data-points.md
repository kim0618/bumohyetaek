# 부모혜택 정책 데이터 포인트 마스터 리스트

> `/refresh` 스킬이 사용하는 변동 가능 정책 수치 데이터베이스.
> 매주 일부 항목을 회전 점검하며, 변경 발견 시 영향 글 일괄 수정.
>
> **/bumo-content로 새 글 추가 시**: 본문에서 검색 패턴 매칭 → 해당 항목의 "영향 글"에 자동 등록.

---

## 가동 원칙

- 마지막 확인일 오래된 순으로 5~7개 픽업
- 각 항목의 공식 출처 WebFetch로 최신값 조회
- 변경 발견 시: 검색 패턴으로 영향 글 식별 → 일괄 수정 → 마스터 리스트 업데이트
- 영향 글 수정 시 동기화 필드: `summary`, `keyPoints`, `sections`, `seoDescription`, `cautionNote`, `effectiveDate`, `updatedAt`
